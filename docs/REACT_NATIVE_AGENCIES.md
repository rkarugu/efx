## 8. Contract Essentials

### **Must Include:**
1. **Scope of Work**
   - Detailed feature list
   - Number of screens
   - API integration requirements
   - Deliverables

2. **Timeline**
   - Milestones with dates
   - Payment schedule
   - Acceptance criteria

3. **Intellectual Property**
   - Code ownership
   - Design assets ownership
   - Third-party licenses

4. **Support & Maintenance**
   - Bug fix period (30-90 days)
   - Maintenance terms
   - Update policy

5. **Payment Terms**
   - Milestone-based payments
   - Escrow (if using platform)
   - Refund policy

6. **Confidentiality**
   - NDA
   - Data protection
   - Non-compete clause

---

## 9. Red Flags to Avoid

### 🚩 Warning Signs
- ❌ No portfolio or GitHub profile
- ❌ Unrealistic timeline (< 6 weeks for full app)
- ❌ Extremely low rates (< $15/hour)
- ❌ 100% upfront payment request
- ❌ No contract or agreement
- ❌ Poor communication
- ❌ No testing process
- ❌ No deployment experience

## 11. Success Tips

### ❌ Don'ts
- ❌ Skip contract/agreement
- ❌ Pay 100% upfront
- ❌ Ignore communication gaps
- ❌ Skip code reviews
- ❌ Rush deployment
- ❌ Ignore testing phase
